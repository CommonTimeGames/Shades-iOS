# Shades-iOS #

A port of Konstantin Uroskin's web Shades game:
http://kuroskin.github.io/Shades/shades.html

Built with SpriteKit for iPhone, iPad, and (eventually) Mac.

To run, open up the Xcode project, click "Run", and boom!
